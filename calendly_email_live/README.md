# Calendly Email Preview

Standalone HTML email mockup inspired by Calendly-style workflow reminders.

## Live Preview

[Open on GitHub Pages](https://joelmuhoho.github.io/email_dev/calendly_email_live/)

## Preview

Open `index.html` directly in a browser or serve the folder with any static file server.

## Editing

Edit `index.html` directly. There is no separate build step in this folder.

> [!NOTE]
> The preview uses hosted image URLs, so it depends on network access unless you replace those assets locally.
