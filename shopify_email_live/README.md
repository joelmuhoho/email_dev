# Shopify Email Preview

Standalone responsive HTML email preview for a Shopify-branded design.

## Live Preview

[Open on GitHub Pages](https://joelmuhoho.github.io/email_dev/shopify_email_live/)

## Preview

Open `index.html` directly in a browser.

## Assets

Local images live in `images/`, and the template also loads hosted fonts and assets.

> [!NOTE]
> There is no build step in this folder. Edit the HTML directly if you need to change the output.
